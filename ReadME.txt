images：原图像
plates：识别得到的车牌
train_dat：存放SVM训练后模型（训练集没有提交）
plate_detect.py：车牌定位实现
svm_train.py：训练SVM模型
text.py：文本检测识别
main.py：整合，直接运行会得到medium三张图片的检测结果，可修改其中的input_dir来得到其它图像的检测结果
